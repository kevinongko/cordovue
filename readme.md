# Cordovue

A sample project to show integration between [Vue.js](https://vuejs.org/) and [Cordova](https://cordova.apache.org/).

## Installation

###Install Node (homebrew)
```sh
$ brew install node
```

### Install Cordova
```sh
$ npm install -g cordova
```

### Build
```sh
$ cd cordovue
$ cordova add platform android
$ npm install
$ npm run build
```
